# Nixx Donation Page

Website donasi personal Nixx dengan tema Cyber Neon + Glassmorphism.

## Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Orbitron, Inter)
- Vercel Static Hosting

## Features
- Particle animation background
- Cursor glow effect
- Loading screen
- Glassmorphism cards
- Neon glow effects
- QRIS donation cards
- Saweria integration
- Counter animations
- Scroll reveal animations
- 3D tilt on QRIS cards
- Fully responsive

## Deploy
Push ke GitHub → Connect di Vercel → Deploy otomatis.
